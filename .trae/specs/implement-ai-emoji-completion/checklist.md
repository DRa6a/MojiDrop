# Checklist

- [ ] 配置系统已实现，配置项包含 API key、请求地址、模型、最大建议数、冷却时间、总开关
- [ ] 配置界面可通过 ModMenu 打开，使用 Cloth Config API 渲染
- [ ] 聊天输入框文本变化被监听，输入空格时触发 AI 请求
- [ ] 请求具备冷却与去重机制，避免请求过快
- [ ] AI 请求服务使用异步 HTTP 客户端，正确构造并解析 OpenAI 兼容请求
- [ ] 网络/API/解析失败时有异常处理，不导致游戏崩溃
- [ ] 颜文字候选以原版建议列表形式展示，支持方向键选择
- [ ] 按 TAB 或点击建议可将颜文字追加到聊天输入框
- [ ] API key 或请求地址为空时跳过请求并提示玩家
- [ ] `./gradlew build` 成功生成 `build/libs/mojidrop-1.0.0.jar`
- [ ] 代码与 spec 文档已提交并推送到远程仓库
